# 🗺️ DSA Learning Roadmap

A topic-by-topic path from fundamentals to advanced. Work top to bottom — each module builds on the previous ones.

---

## 01 · Foundations & Complexity
The language you use to reason about every algorithm.
- What is an algorithm & a data structure
- Big-O, Big-Ω, Big-Θ notation
- Time complexity analysis
- Space complexity analysis
- Common complexity classes: O(1), O(log n), O(n), O(n log n), O(n²), O(2ⁿ)

## 02 · Arrays & Strings
- Array traversal & in-place operations
- Prefix sums
- Two-pointer technique
- Sliding window
- Common string manipulation patterns

## 03 · Searching Algorithms
- Linear search
- Binary search (and its variants: first/last occurrence, search in rotated array)
- Time complexity trade-offs

## 04 · Sorting Algorithms
- Bubble, selection, insertion sort
- Merge sort (divide & conquer)
- Quick sort (partitioning)
- Heap sort
- Counting / radix sort (non-comparison)
- Stability & when to use which

## 05 · Recursion & Backtracking
- How the call stack works
- Base case vs recursive case
- Recursion → iteration
- Backtracking: subsets, permutations, N-Queens, Sudoku

## 06 · Linked Lists
- Singly linked list
- Doubly linked list
- Circular linked list
- Reverse, detect cycle (Floyd's), merge, middle node

## 07 · Stacks & Queues
- Stack (LIFO) & applications (balanced parentheses, undo)
- Queue (FIFO)
- Deque & circular queue
- Monotonic stack/queue

## 08 · Hashing & Hash Tables
- Hash functions & collisions
- Chaining vs open addressing
- Using `Map` and `Set` in JavaScript
- Frequency counting patterns

## 09 · Trees & Binary Search Trees
- Tree terminology
- Binary tree traversals (pre/in/post-order, level-order)
- Binary Search Tree: insert, search, delete
- Height, depth, balanced trees (intro to AVL)

## 10 · Heaps & Priority Queues
- Min-heap & max-heap
- Heapify, insert, extract
- Priority queue implementation
- Top-K problems

## 11 · Graphs
- Representations: adjacency list vs matrix
- BFS & DFS
- Topological sort
- Shortest path: Dijkstra, Bellman-Ford
- Minimum spanning tree: Kruskal, Prim

## 12 · Greedy Algorithms
- The greedy-choice property
- Activity selection, interval scheduling
- Huffman coding
- When greedy fails (and why DP is needed)

## 13 · Dynamic Programming
- Overlapping subproblems & optimal substructure
- Memoization (top-down) vs tabulation (bottom-up)
- Classic problems: Fibonacci, knapsack, LCS, coin change, edit distance

## 14 · Tries & Advanced Topics
- Trie (prefix tree)
- Union-Find / Disjoint Set Union
- Segment tree / Fenwick tree
- Bit manipulation tricks
