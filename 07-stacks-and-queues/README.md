# 07 · Stacks & Queues

Two restricted-access structures that show up everywhere: parsing, scheduling, traversal, undo/redo.

## 📖 Suggested lessons
- `lesson-01-stack` — LIFO, push/pop, applications (balanced brackets, undo)
- `lesson-02-queue` — FIFO, enqueue/dequeue
- `lesson-03-deque-and-circular-queue` — double-ended and ring buffers
- `lesson-04-monotonic-stack` — next greater element and friends

## 🎯 You'll be able to
- Implement a stack and queue (array-based and linked-list-based)
- Recognize when a problem needs LIFO vs FIFO ordering

## 🧩 Practice ideas
Valid Parentheses · Min Stack · Implement Queue using Stacks · Daily Temperatures
