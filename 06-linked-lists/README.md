# 06 · Linked Lists

Your first pointer-based structure. Great for practicing careful reference manipulation.

## 📖 Suggested lessons
- `lesson-01-singly-linked-list` — node, head, traversal, insert, delete
- `lesson-02-doubly-linked-list` — prev + next pointers
- `lesson-03-circular-linked-list` — tail points back to head
- `lesson-04-classic-problems` — reverse, detect cycle (Floyd's), find middle, merge two lists

## 🎯 You'll be able to
- Build a linked list class in JavaScript
- Manipulate pointers without losing the rest of the list
- Detect cycles with the fast/slow pointer trick

## 🧩 Practice ideas
Reverse Linked List · Linked List Cycle · Merge Two Sorted Lists · Remove Nth Node From End
