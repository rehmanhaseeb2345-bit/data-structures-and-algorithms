# 12 · Greedy Algorithms

Make the locally optimal choice at each step. Simple and fast when it works — the skill is proving that it does.

## 📖 Suggested lessons
- `lesson-01-greedy-fundamentals` — the greedy-choice property & optimal substructure
- `lesson-02-interval-problems` — activity selection, interval scheduling, merge intervals
- `lesson-03-classic-greedy` — jump game, gas station, Huffman coding

## 🎯 You'll be able to
- Identify when a greedy approach is provably correct
- Recognize when greedy fails and DP is required instead
