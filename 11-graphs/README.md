# 11 · Graphs

The most general structure — nodes and edges. Trees and linked lists are special cases. Master traversal first, then shortest paths.

## 📖 Suggested lessons
- `lesson-01-graph-representations` — adjacency list vs adjacency matrix
- `lesson-02-bfs-and-dfs` — the two core traversals
- `lesson-03-topological-sort` — ordering a DAG
- `lesson-04-shortest-paths` — Dijkstra, Bellman-Ford
- `lesson-05-minimum-spanning-tree` — Kruskal, Prim

## 🎯 You'll be able to
- Model a problem as a graph
- Choose BFS vs DFS appropriately
- Compute shortest paths on weighted graphs
