# 09 · Trees & Binary Search Trees

Hierarchical data. Traversals and BST operations are the foundation for heaps, tries, and much of interviewing.

## 📖 Suggested lessons
- `lesson-01-tree-terminology` — root, node, leaf, height, depth
- `lesson-02-binary-tree-traversals` — pre/in/post-order + level-order (BFS)
- `lesson-03-binary-search-tree` — insert, search, delete
- `lesson-04-tree-problems` — height, diameter, is-balanced, lowest common ancestor

## 🎯 You'll be able to
- Traverse a tree both recursively and iteratively
- Implement a BST and reason about why traversal order matters

## 🧩 Practice ideas
Maximum Depth of Binary Tree · Invert Binary Tree · Validate BST · Level Order Traversal
