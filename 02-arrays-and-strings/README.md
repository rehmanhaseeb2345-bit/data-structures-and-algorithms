# 02 · Arrays & Strings

The most common data type in interviews. Master the core patterns and a huge chunk of problems become approachable.

## 📖 Suggested lessons
- `lesson-01-array-basics` — traversal, insertion, deletion, in-place edits
- `lesson-02-prefix-sums` — precompute running totals for fast range queries
- `lesson-03-two-pointers` — opposite-end and fast/slow pointer techniques
- `lesson-04-sliding-window` — fixed and variable-size windows
- `lesson-05-string-patterns` — reversing, anagrams, palindromes

## 🎯 You'll be able to
- Manipulate arrays and strings in place with O(1) extra space
- Reach for the right pattern (two pointers vs sliding window) per problem

## 🧩 Practice ideas
Two Sum · Valid Palindrome · Best Time to Buy and Sell Stock · Longest Substring Without Repeating Characters
