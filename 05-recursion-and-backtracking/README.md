# 05 · Recursion & Backtracking

Thinking recursively is a prerequisite for trees, graphs, and dynamic programming. Backtracking is systematic brute force done well.

## 📖 Suggested lessons
- `lesson-01-recursion-basics` — base case, recursive case, the call stack
- `lesson-02-recursion-patterns` — factorial, power, Fibonacci, array recursion
- `lesson-03-backtracking` — subsets, permutations, combinations
- `lesson-04-classic-backtracking` — N-Queens, Sudoku solver, word search

## 🎯 You'll be able to
- Trace the call stack for a recursive function
- Convert simple recursion to iteration
- Structure a backtracking solution: choose → explore → un-choose

## 🧩 Practice ideas
Subsets · Permutations · Combination Sum · N-Queens · Generate Parentheses
