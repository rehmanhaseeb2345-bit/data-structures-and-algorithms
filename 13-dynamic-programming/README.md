# 13 · Dynamic Programming

Solve big problems by combining answers to overlapping subproblems. The most feared interview topic — demystified by lots of practice.

## 📖 Suggested lessons
- `lesson-01-dp-fundamentals` — overlapping subproblems & optimal substructure
- `lesson-02-memoization` — top-down recursion + cache
- `lesson-03-tabulation` — bottom-up iteration
- `lesson-04-1d-dp` — climbing stairs, house robber, coin change
- `lesson-05-2d-dp` — longest common subsequence, edit distance, knapsack

## 🎯 You'll be able to
- Spot a DP problem (choices + overlapping subproblems)
- Convert a recursive solution into memoized, then tabulated form

## 🧩 Practice ideas
Climbing Stairs · House Robber · Coin Change · Longest Common Subsequence · 0/1 Knapsack
