# 10 · Heaps & Priority Queues

A tree stored in an array that always gives you the min or max in O(log n). The go-to for "top-K" and scheduling problems.

## 📖 Suggested lessons
- `lesson-01-heap-fundamentals` — complete binary tree, array representation
- `lesson-02-heap-operations` — heapify, insert (sift-up), extract (sift-down)
- `lesson-03-priority-queue` — building a PQ on top of a heap
- `lesson-04-top-k-problems` — k largest/smallest, merge k lists

## 🎯 You'll be able to
- Implement a min-heap and max-heap from scratch
- Solve top-K problems in O(n log k)

## 🧩 Practice ideas
Kth Largest Element · Top K Frequent Elements · Merge k Sorted Lists · Find Median from Data Stream
